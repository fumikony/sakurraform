# Sakurraform

Manage Infrastructure from Code with Sakura no Cloud

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'sakurraform'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install sakurraform
